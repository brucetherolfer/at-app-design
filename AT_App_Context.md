# Alexander Technique Reminder App — Project Context

## Goal
Phone buzz notifications to prompt Alexander Technique awareness throughout the day.

## Current Status (v1 — working)
- iOS: Scriptable app (free)
- 100 AT prompts in Apple Reminders, "Alexander Technique" list
- Saturday & Sunday only, 9am-9pm, ~7 min fixed intervals

## How to Run Again
1. Open Scriptable on iPhone
2. Open AT_Reminders_Script.js
3. Tap Run — creates 200 reminders for next Sat/Sun

## Planned Enhancements
1. Randomize intervals (easy script change)
2. Full week with massage schedule blackout windows:
   - Mon/Tue/Thu/Fri: block massage session times (confirm times)
   - Wednesday: most open day
3. Dedicated app if concept proves out
